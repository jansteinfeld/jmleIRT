# jmleIRT: Joint Maximum Likelihood Estimation of the Rasch Model

Implements joint maximum likelihood estimation (JMLE) for item
parameters in the Rasch model (1PL). Provides methods for bias
correction of item parameter estimates and weighted likelihood
estimation (WLE) of person parameters.
