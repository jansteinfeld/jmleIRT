# Summary for Bias Corrected JMLE Estimates

Print a summary of bias-corrected item difficulty and person ability
estimates.

## Usage

``` r
# S3 method for class 'biasCorrection'
summary(object, ...)
```

## Arguments

- object:

  Object of class `"biasCorrection"` containing `corrected_b` and
  `corrected_theta`.

- ...:

  Additional arguments (currently unused).

## Value

The input `object`, invisibly.
